Fitur yang Mengubah Niat Baik Jadi Aksi Nyata

Teknologi canggih yang memudahkan setiap langkah wakaf Anda

Pengelolaan Data Muzakki

Mencatat data para muzakki (wajib zakat) dan juga donatur infaq, shadaqah maupun wakaf.

Pengelolaan Data Mustahik

Mencatat data para mustahiq (penerima zakat) dengan pengelompokan berdasarkan asnaf zakat.

Pengelolaan Data Umat

Gabungan dari data muzakki dan data mustahiq dan juga data jama'ah lainnya.

Pendistribusian ZISWAF

Mempercepat proses perhitungan pendistribusian zakat.

Monitoring Pengelolaan

Memberikan kemudahan kepada para pengurus dan stakeholders masjid untuk mengelola secara online.

Laporan

Rekap hasil laporan data Muzakki, Zakat non Fitrah, data Asnaf, data Cluster dan data Pembayaran Zakat.

Download Data

Mempermudah pengurus melakukan rekap data dan juga penyajian informasi berbasis cetak.

Pengelolaan Pengguna

Dapat diakses oleh beberapa orang sekaligus.

Print Struk

Memberikan bukti tanda terima ziswaf dengan melakukan print.
